# Elegant Beauty Style Suite

Premium online, door-to-door salon booking platform for hairstyles, African plaiting, nails, skincare, children's hair care, wedding preparation, and interview grooming.

## Stack

- Frontend: React, Vite, TypeScript, Tailwind CSS, Framer Motion, React Router, React Hook Form, TanStack Query, Axios, PWA
- Backend: Node.js, Express, TypeScript, PostgreSQL, Drizzle ORM, JWT, Zod
- Deployment: Railway-ready config, environment-driven services, seed and migration scripts

## Quick Start

```bash
npm install
cp .env.example .env
npm run db:migrate
npm run db:seed
npm run dev
```

Frontend runs on `http://localhost:5173`.
Backend runs on `http://localhost:4000`.
